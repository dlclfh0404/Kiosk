# Kiosk
java를 이용하여 만든 Kiosk 프로그램입니다.
## 프로그램 설명
- SceneBuilder프로그램을 이용하여 fxml파일을 작성합니다
- Eclipse 프로젝트 내에 Controller파일을 작성하고 On Actionr값을 설정하고 SceneBuilder 를 재실행후 controller를 연결을 하고
- 저장 한 후에 실행을 시켜줍니다.
